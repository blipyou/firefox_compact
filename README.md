# firefox_compact
开启Firefox紧凑模式

自 Firefox 89 版本更新（代号 Proton UI）之后，Mozilla 为了适应触屏设备，把标签页变成了“悬浮方块”设计，确实**变高了，也变胖了**，很多老用户（尤其是笔记本小屏幕用户）都觉得浪费屏幕纵向空间。

既然你用了 Firefox，就没有“改不了”的东西。这里有**浅、中、深**三种方案，帮你把标签栏压扁。

---

### 方案一：开启官方隐藏的“紧凑模式” (Compact Mode)
**难度：⭐**
**效果：能变矮一点，最安全稳定。**

Mozilla 官方其实保留了一个“紧凑模式”，但默认把它藏起来了。我们需要手动把它挖出来。

1.  在地址栏输入 `about:config`，回车，点击“接受风险并继续”。
2.  在搜索框输入：`browser.compactmode.show`
3.  点击右侧的切换按钮（双向箭头），把它的值从 `false` 变为 `true`。
4.  现在，右键点击标签栏空白处 -> 选择 **“定制工具栏…” (Customize Toolbar...)**。
5.  在底部找到 **“密度” (Density)** 下拉菜单。
6.  你会发现多了一个 **“紧凑” (Compact)** 选项，选中它，点击“完成”。

你会发现标签栏瞬间收缩了一圈，不再那么臃肿了。

---

### 方案二：使用 CSS 代码暴力修改 (userChrome.css)
**难度：⭐⭐⭐**
**效果：想多矮就多矮，甚至能回到旧版外观。**

如果方案一还不够矮，或者你讨厌那个“悬浮”的设计，我们可以用之前提到的 `userChrome.css` 来重新定义界面。

**第一步：开启 CSS 支持**
1.  地址栏输入 `about:config`。
2.  搜索 `toolkit.legacyUserProfileCustomizations.stylesheets`。
3.  把它设置为 `true`。

**第二步：创建样式文件**
1.  地址栏输入 `about:support` (故障排除信息)。
2.  找到 **“配置文件夹” (Profile Folder)** 这一行，点击旁边的 **“打开文件夹”**。
3.  在这个文件夹里，新建一个文件夹，命名为 `chrome`（全小写）。
4.  进入 `chrome` 文件夹，将仓库中的 `userChrome.css`放进去。

**第三步：重启 Firefox**
* 保存文件，关闭并重新打开 Firefox。你会发现标签栏已经被“压扁”了。你可以随意修改代码里的 `25px` 来调整高度。

---

### 方案三：彻底干掉顶栏（配合垂直标签页）
**难度：⭐⭐⭐⭐**
**效果：0高度，极致视野。**

如果你刚才安装了我推荐的 **Sidebery**（侧边栏垂直标签页），那么你头顶的横向标签栏其实是多余的。不如直接把它隐藏掉，让网页内容顶天立地。

同样是在 `userChrome.css` 里，加入这行代码：

```css
/* 隐藏顶部标签栏 */
#TabsToolbar {
  visibility: collapse !important;
}
/* 如果你想保留红绿灯(Mac)或最小化关闭按钮(Win)，代码会更复杂一点，
   但简单的 collapse 是最直接的隐藏 */
```

**警告：** 这样做之后，你**只能**靠侧边栏（Sidebery）来切换标签页了。这就是很多硬核极客的终极形态：**左侧树状管理，右侧全屏浏览。**

### 建议
先试**方案一**。如果觉得还不够紧凑，再尝试**方案二**。这是 Firefox 最迷人的地方：**只要你懂一点代码，它就是你的橡皮泥。**
